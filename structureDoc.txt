Page: Disputes (النزاعات)

- Sidebar
- Header
- Main

  - SummaryCards

    - Card (نزاعات مفتوحة)
    - Card (قيد المراجعة)
    - Card (نزاعات محلولة)

  - Table (قائمة النزاعات)

    - FilterBar (Select الحالة)
    - Columns: ID / المستأجر / المؤجر / المعدة / مبلغ النزاع / تاريخ الفتح / الحالة / إجراء

  - ReviewPage (مراجعة النزاع)

    - Header (عنوان + ID + StatusBadge + BackButton)

    - Column1: بيانات الطرفين

      - DeliveryPhotos
      - ReturnPhotos
      - ContractButton

    - Column3: اتخاذ القرار

      - DisputeAmount
      - RadioGroup (قبول الخصم / رفض الخصم / تعديل القيمة)
      - AdjustedAmountInput
      - Textarea (ملاحظة إدارية)
      - WarningAlert

Page: Settings (الإعدادات)

- Sidebar
- Header
- Main

  - Tabs (إدارة الأدوار والصلاحيات / الأمان والمصادقة)

  - Tab: إدارة الأدوار والصلاحيات

    - AdminsTable

      - Header (عنوان + AddButton)
      - Columns: الاسم / البريد الإلكتروني / الدور (Select) / إجراء (تعديل / حذف)

    - PermissionsMatrix

      - Columns: الصلاحية / Super Admin / Support / Finance
      - Rows: إدارة المستخدمين / الإشراف المالي / إدارة النزاعات / Audit Log / إدارة المعدات

  - Tab: الأمان والمصادقة

    - SecuritySettings

      - Toggle (المصادقة متعددة العوامل 2FA)
      - Select (مدة انتهاء الجلسة)

    - ActiveSessions

      - SessionCard (الجهاز / IP / الحالة / إنهاء الجلسة)
